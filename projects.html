Bookmark

A fun little day-long project with the aim of familiarising myself with the Kotlin programming language and also creating something that I, an avid reader, will use. As someone who reads multiple books at once it can sometimes be difficult to track my page in all of them, and all of the extra fluff of Goodreads can be somewhat annoying. This app simply allows you to add your book and keep track of the page number.

Crossword Solver

I wrote this when I was doing my A-levels and it's my first published app. This app has three functions (whether the functions function is up for debate). Firstly allows users to input words with missing letters to get all possible suggestions. It also can solve anagrams. Finally it allows you to search a word definition by clicking on a result.
